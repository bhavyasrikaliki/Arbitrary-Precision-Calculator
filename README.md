# Arbitrary Precision Calculator

An Arbitrary Precision Calculator implemented in C using Data Structures to perform arithmetic operations on very large integers that exceed standard data type limits.

---

## Features
- Supports large integer arithmetic beyond built-in data types
- Addition of large numbers
- Subtraction of large numbers
- Multiplication of large numbers
- Division of large numbers
- Uses linked lists for digit-wise operations

---

## Technologies Used
- C Programming
- Data Structures
- Linked Lists
- Algorithms

## How to Compile and Run

### Compile
```bash
gcc *.c -o apc
```

### Run
```bash
./apc
```
